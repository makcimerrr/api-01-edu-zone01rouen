# API 01 Edu — Zone01 Rouen

REST API construite avec **Deno + Oak**, qui expose les données pédagogiques
de Zone01 Rouen au-dessus de la plateforme **01 Edu** : utilisateurs,
promotions, projets, configuration Discord, intégration Gitea, etc.

## En une requête

```bash
export API="https://api-zone01-rouen.deno.dev"
curl $API/api/v1/promotions
```

## Pour qui ?

- **Outils internes** Zone01 Rouen : dashboards de progression, exports.
- **Bots Discord** consommant `/discord/*` et `/discord-users/*`.
- **Scripts** d'agrégation de données (Gitea, Toad, Piscine).
