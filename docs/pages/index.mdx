# API 01 Edu — Zone01 Rouen

Bienvenue dans la documentation de l'API REST construite avec **Deno + Oak** qui
expose les données pédagogiques de Zone01 Rouen (utilisateurs, promotions,
projets, configurations Discord, etc.) au-dessus de la plateforme **01 Edu**.

## En une requête

```bash
curl https://api.zone01normandie.org/api/v1/promotions
```

## Pour qui ?

- **Outils internes** Zone01 Rouen (dashboards, rapports de progression).
- **Bots Discord** consommant `/discord/*` et `/discord-users/*`.
- **Scripts d'export** vers Gitea, Toad, etc.
