# Support

## Bug ou demande d'évolution

Ouvrir une issue sur GitHub :
https://github.com/makcimerrr/api-01-edu-zone01rouen/issues/new

Inclure :
- Le path appelé + méthode HTTP
- Le body envoyé (si applicable)
- La réponse reçue (status + body)
- Le `x-deno-ray` du header de réponse (utile pour le diagnostic Deno
  Deploy)

## Sécurité

Pour signaler une vulnérabilité, **ne pas ouvrir d'issue publique**.
Contacter directement le mainteneur (cf. profil GitHub @makcimerrr).

## Status & monitoring
