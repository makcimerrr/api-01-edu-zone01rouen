# Changelog

Toutes les évolutions notables, par ordre antéchronologique. Les **breaking
changes** sont marqués 🔴.

## 2026-05-02

- ➕ Endpoint [`GET /api/v1/projects/catalog`](/reference/projects) :
  catalogue complet des projets (tronc commun + optionnels + additionnels
  + spécialités). Comble l'absence d'une route exposant tous les projets
  toutes catégories confondues.
- 🚀 **Documentation Nextra** publiée sous `/docs/`.
- 🔁 La route `/` redirige désormais (`302`) vers `/docs/`.
- ➕ Pages : Tous les endpoints, Glossaire, CORS, Versioning, Support, SDKs.

## 2026-04 (commit `4e419ad`)

- ➕ Endpoint [`GET /api/v1/specialties`](/reference/specialty) (catalogue des spécialités).
