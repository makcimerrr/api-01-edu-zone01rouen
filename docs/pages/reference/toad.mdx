# Toad

Sessions Toad (challenges techniques de pré-sélection / recrutement Zone01).
Renvoie pour chaque candidat la liste des jeux passés et son **meilleur
niveau atteint** par jeu.

> Source : `src/api/v1/toad.ts`. Données lues via GraphQL 01 Edu
> (`toad_sessions`).

---

## Lister les sessions Toad

<Endpoint method="GET" path="/api/v1/toad/sessions" source="src/api/v1/toad.ts" />

### Réponse `200`

```json
{
  "toad_sessions": [
    {
      "candidate": { "login": "alice" },
      "games": [
        {
          "name": "Find Your Way",
          "results": [{ "level": 7 }]
        },
        {
          "name": "Logic Maze",
          "results": [{ "level": 4 }]
        }
      ]
    }
  ]
}
```

> `results` est limité à **1 entrée** triée par `level` décroissant — c'est
> donc toujours le **meilleur niveau** atteint pour ce jeu.

### Erreurs

| Code | Cause                  |
| ---- | ---------------------- |
| 500  | Erreur GraphQL 01 Edu  |

### Exemple

```bash
curl $API/api/v1/toad/sessions
```
